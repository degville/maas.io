Metal as a Service

Metal as a Service (MAAS) brings the language of the cloud to physical servers. It makes it easy to set up the hardware on which to deploy any service that needs to scale up and down dynamically; a cloud being just one example.

Get started

Why use MAAS?

  • Clarity/Visibility

    Consolidated overview of the infrastructure

    (BMCs, machines, networks, availability zones) and deployments (OS, owner, workload)

  • Elasticity

    Manage and deploy physical machines just like a cloud

  • Speed

    Deploy Ubuntu workloads in less than 3 minutes

  • Flexibility

    Deploy multiple operating systems

    (Ubuntu, CentOS, RHEL, Windows, SLES, custom images)
